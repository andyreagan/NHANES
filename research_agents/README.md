# Research Agent Investigation Log

This directory documents 16 parallel AI research agent investigations into
reproducing the Levine et al. 2018 PhenoAge Gompertz coefficients. Each agent
explored a specific hypothesis for why our fitted coefficients diverged from
the published values.

## Context

Our initial PhenoAge reproduction matched the sample size (n=9,926) but
coefficients differed by ~25% on average, with glucose the worst at ~47% off.
These agents systematically eliminated hypotheses until the root cause was
identified: **irreducible data composition differences**, not implementation
errors.

## Investigation Arc

### Round 1: Initial hypotheses (2019 mortality file)

| Agent | File | Hypothesis | Verdict |
|-------|------|-----------|---------|
| 1 | `results_agent1_flexsurv.md` | Gompertz parameterization differs between Python and R's flexsurv | ❌ Rejected — parameterization is not the issue |
| 2 | `results_agent2_glucose.md` | Glucose unit conversion, variable choice (SGP vs G1P), or nonlinear transform | ❌ Rejected — all give ~0.075, systematically off |
| 3 | `results_agent3_optimizer.md` | MLE optimizer stuck in local minimum or has math error | ⚠️ Found missing `log(γ)` term, but fixing it didn't close the gap |
| 3b | `results_agent3_robust.md` | Numerical stability issues in log-likelihood | ✅ Documented correct formula; optimizer confirmed correct |
| 4 | `results_agent4_sample.md` | Death count discrepancy (1,885 vs 1,846) | ✅ Identified mortality file version as cause (2019 vs 2015) |

### Round 2: With real 2015 mortality file

| Agent | File | Hypothesis | Verdict |
|-------|------|-----------|---------|
| 5 | `results_agent5_2015_deaths.md` | Different mortality exclusion codes or follow-up caps | ✅ Best death match: excl {4,10} → 1,821 deaths (25 short) |
| 6 | `results_agent6_2015_glucose.md` | Glucose-specific data issues (fasting, serum vs plasma, transforms) | ❌ Rejected — no glucose hypothesis explains the gap |

### Round 3: R flexsurvreg on identical data

| Agent | File | Hypothesis | Verdict |
|-------|------|-----------|---------|
| 9 | `results_agent9_r_mort_variants.md` | Test multiple mortality filter variants in R | ✅ Matched death count (1,853 vs 1,846) but coefficients still off |
| 10 | `results_agent10_r_5sd_variants.md` | R vs Python 5SD outlier removal implementation | ❌ Rejected — R and Python give identical 5SD results |
| 11 | `results_agent11_r_completeness.md` | Different completeness filters in R | ❌ All filters give 22-37% avg diff; glucose always ~47% off |
| 12 | `results_agent12_r_weights.md` | Survey weights (WTPFEX6) | ⚠️ Fixes intercept (-19.85 vs -19.91) but glucose only +1.5% |
| 13 | `results_agent13_r_winsorize.md` | Winsorizing biomarker outliers | ✅ Excluding problem variable outliers → 18.9% avg diff |
| 14 | `results_agent14_r_remove_extremes.md` | Systematic removal of extreme observations | ⚠️ Marginal improvements only |
| 15 | `results_agent15_r_glucose_focus.md` | Cox vs Gompertz; univariate glucose coefficient | ✅ Univariate glucose = 0.205 (5% off!); multivariate = 0.102 (47% off) |
| 16 | `results_agent16_r_multivar_caps.md` | Clinical range caps and statistical winsorization | ✅ **Best result: 2SD winsorization → 15.4% avg diff** |

## Key Conclusions

1. **No implementation bug exists.** R flexsurvreg matches our Python MLE to 0.01%.
2. **The data differs.** Our MLE achieves 100 LL-units better than Levine's
   published coefficients on our data — the coefficients are correct for our data.
3. **Winsorizing is the key.** 2SD winsorization + glucose cap at 6.0 mmol/L brings
   avg coefficient difference from 24.6% → 11.3%, with 6/10 coefficients within 5%.
4. **WBC is irreducible.** The WBC coefficient (44% off) likely reflects CBC
   instrument calibration differences in the original data files.
5. **It doesn't matter for practice.** All model variants produce PhenoAge scores
   that correlate r > 0.99 with each other and predict mortality equivalently
   (C-index within 0.002 on held-out NHANES IV).
